Two pipelines. One private brain for your documents.

Every document flows through an Ingestion Pipeline that reads, understands and indexes it. Every search flows through a Retrieval Pipeline that finds the most relevant passages using two complementary engines fused together. Both run on a single GPU server.

The AI Brain

Six specialised models — each doing one job brilliantly.

DocuSync deliberately uses small, focused models instead of one giant LLM. They're faster, fully local, and auto-detect the best device (CUDA → MPS → CPU). GPU inference is serialised by a semaphore so models never collide in memory.

Embeddings · Bi-encoder

BGE-base-en-v1.5

Converts every chunk and every query into a 768-dimensional vector so meaning can be compared by cosine distance. The backbone of semantic search.
768-dimBAAICUDA
Reranking · Cross-encoder

GTE-Reranker-ModernBERT

Reads the query and a candidate passage together to judge true relevance — far more accurate than vector distance. Applied to the top candidates only.
max_len 256Alibaba-NLPtop 15–30
Classification · Zero-shot NLI

DeBERTa-v3-large-MNLI

Labels each document across 4 dimensions — Subject, Field, Document Type, Methodology — without any training data, using natural-language inference at a 0.35 confidence threshold.
zero-shot4 dimensionsthr 0.35
Summarisation · Seq2seq

DistilBART-CNN-12-6

Produces a short abstractive summary that rephrases what each document is about — shown in the inspector's AI Summary panel.
abstractivedistilled BARTCUDA
OCR · Vision ensemble

PaddleOCR + Tesseract

When a PDF page has no extractable text (scanned image), the page is rasterised and read by an OCR ensemble so even scans become fully searchable.

Pipeline 2 · Retrieval

Hybrid search: two engines, fused, then re-judged.

A keyword query alone misses meaning; a semantic query alone misses exact terms. DocuSync runs both in parallel, fuses them with Reciprocal Rank Fusion, then a cross-encoder re-judges the finalists. Type a query below and watch it happen.

Deployment & Reliability

Built to stay up — and stay private.

The whole platform runs as managed services on a single GPU machine. Everything auto-starts on boot in the correct order and auto-restarts on failure.

🟢

Always-On Services

systemd

Three managed units: Tailscale (private mesh network), DocuSync (the FastAPI app), and Cloudflare Tunnel (public URL). Each restarts automatically if it ever crashes.

Restart=alwaysenabled on bootBindsTo=docusync
🔐

Safe Shutdown

operator guard

The server can't be powered down by accident. A guarded safe-shutdown command requires the operator to type a confirmation phrase, then gracefully drains and stops every service before power-off.

confirm phrasegraceful drainTimeoutStopSec=30
🎮

GPU Acceleration

NVIDIA RTX 4080

All six models auto-detect and run on CUDA. A semaphore serialises heavy inference so the GPU is never over-subscribed, keeping ingestion stable and search warm at ~90 ms.

CUDA → MPS → CPU1-at-a-time inference~90ms search

Access Anywhere, Privately

Tailscale + Cloudflare

Team members reach the platform over an encrypted Tailscale mesh by private IP, while a Cloudflare Tunnel can expose a shareable HTTPS link for demos — with documents never leaving your server.
